# 项目初衷

青史，一个历史性人物和事件的镌刻之地。

不如百科全面，亦无百科冗长；

寥寥数百字，青史留世间。

知来日方长，愿事事皆好。

本站由李文亮医生去世引发思考，进而动手搭建。


# 涉及内容

青史，有名垂青史，也有遗臭万年。什么样的事件和人物值能在中华民族的历史长河中激起浪花，就应该永刻青史。



# 入史要求：**

- 事件或人物受到全民关注、讨论反思
- 事件或人物必须客观事实、用语中立
- 事件或人物促进制度变革、社会进步
- 伟大的事件或人物
